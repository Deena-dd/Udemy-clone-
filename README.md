This project is a pixel-perfect, static replica of the Udemy homepage front-end. It is designed as a pure-styling challenge to demonstrate proficiency in modern web architecture without using external frameworks or JavaScript. 
Key Features
* Fully Responsive Design: Uses media queries to ensure a seamless experience across mobile, tablet, and desktop screens.
* Modern CSS Layouts: Built using Flexbox and CSS Grid for complex alignment, such as the course grid and category sections.
* Semantic HTML5: Structured with accessibility in mind, utilizing semantic tags like <header>, <nav>, <main>, and <footer>.
* Interactive Components: Features CSS-only hover effects, smooth transitions, and polished UI components (e.g., course cards and search bars).
